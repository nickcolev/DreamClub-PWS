29-Jan-15 18:18, Sign key: Generated private key to sign the app as described here.

keytool -genkey -v -keystore PWS.keystore -alias PWS -keyalg RSA -keysize 2048 -validity 1825

Before build release, added to ant.properties

key.store=PWS.keystore
key.alias=PWS

Added ant.properties to the git. Successfuly built pws.apk. The technology was:

ant release							Internally calls 'jarsigner' (prompt for password). Creates 'bin/pws-release.apk'
jarsigner -verify -verbose -certs bin/pws-release-unaligned.apk	Verify the apk is signed
zipalign -v 4 bin/pws-release-unaligned.apk bin/pws.apk		Final apk for distribution
